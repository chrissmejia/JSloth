## First steps

* Create config.json file (A sample_config.json is provided).

* Install dependencies:

```bash
npm install
```

* Run the tests:

```bash
npm test
```

* Compile the code:

```bash
grunt
```

* Run the server:

```bash
npm start
```

### Watch for changes

```bash
grunt watch
```
